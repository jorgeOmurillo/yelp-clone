# Yelp Clone

A Yelp Clone app made with Node.js, MongoDB, and EJS.<br/>
[https://jm-yelp-clone.herokuapp.com/](https://jm-yelp-clone.herokuapp.com/)

### Installing

To install the project dependencies, run:

```
npm install
```

To run the server:
```
npm start
```

## Built With

- Node.js
- MongoDB
- EJS
